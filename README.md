# 🧬 KRAS G12C Clinical Evidence Engine

An automated data extraction, statistical auditing, and visualization platform built to parse oncological clinical trial endpoints from ClinicalTrials.gov.

## 🏗️ Engineering Architecture & Milestone 1 Deliverables
* **Data Harvesting Pipeline:** Programmatic ingestion engine syncing live endpoints from the ClinicalTrials.gov v2 API.
* **Statistical Trust Engine:** Implemented standard clinical trial Hazard Ratio (HR) statistical power protocols, validating study enrollment sizes against a required 447-patient baseline.
* **Data Normalization:** Programmatic entity normalization mapping obscure chemical pipeline terms into standard names.
* **High-Fidelity Dashboard Layout:** Standalone HTML5 dashboard rendering raw trial objects into color-coded, interactive Knowledge Cards.
